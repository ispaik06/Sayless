# Platform logo assets

Place only official, permitted brand assets in this directory.

Expected filenames used by the website:

- `kakaotalk.svg` or update the site data after confirming an officially permitted KakaoTalk asset.
- `instagram.svg`
- `discord.svg`
- `slack.svg`

Use the official brand resource pages before adding files:

- Kakao Developers Design Guide: https://developers.kakao.com/docs/latest/en/kakaologin/design-guide
- Instagram Brand Resources: https://www.meta.com/brand/resources/instagram/instagram-brand/
- Instagram Icons: https://www.meta.com/brand/resources/instagram/icons/
- Discord Branding: https://discord.com/branding
- Slack Media Kit: https://slack.com/media-kit

Do not redraw, recolor, distort, or reconstruct logos. If a logo file is absent, the website falls back to a text-only app name card.
